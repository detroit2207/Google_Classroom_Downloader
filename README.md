# Google_Classroom_Downloader
An open source electron app helps to download and organize all of your uni's materials by just using the classrooms link


**Installation steps:**  

  Step 1:<br>
     &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; If you have not installed python yet, open powershell and paste the following command and install python.
    
    winget install 9PNRBTZXMB4Z

  Step 2:<br>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Install pre-requisites by pasting the following commands in powershell or command prompt.

    
        pip install cryptography
        pip install google-auth google-auth-oauthlib
        pip install google-api-python-client
        pip install python-dotenv  
       
  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;press enter and wait for the last command to complete its execution and then close the command prompt.

  Step 3:<br>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Go to our releases page and download the setup file and run it.

  Step 4:<br>
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Now the app will be opened and now you can paste any classroom link and download its files.
